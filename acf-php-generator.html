Что входит

PHP-регистрация ACF без ручной сборки массива

Страница ведет в preset, который готовит структуру для acf_add_local_field_group: group key, Location rules, field names и стабильные ключи для functions.php.

  • Группа Content Fields
  • PHP registration array
  • Location rules и стабильные field keys

Как использовать

  1. 1

    Откройте генератор с готовой группой Content Fields и PHP export.

  2. 2

    Переименуйте группу и поля под свой тип записи, шаблон или страницу.

  3. 3

    Скопируйте PHP-массив и подключите его в проект WordPress.

acf_add_local_field_group([
  'key' => 'group_acf_php_generator',
  'title' => 'ACF PHP генератор',
  'fields' => [
    // fields from generator
  ],
]);

Вопросы по ACF PHP генератор

Для чего нужен этот генератор?

Нужен быстрый PHP export без ручной сборки массива полей.

Что получится на выходе?

Content Fields preset, PHP-массив для acf_add_local_field_group, Location rules и код для functions.php.

Можно ли доработать поля после генерации?

Да, генератор дает стартовую структуру. Поля можно переименовать, удалить, вложить или дополнить перед экспортом.
