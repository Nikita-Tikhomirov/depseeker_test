Что входит

Поля первого экрана без ручной сборки

Страница показывает, какие ACF-поля нужны для hero-блока: текст, CTA, медиа и доверительные маркеры. Из нее пользователь сразу попадает в генератор с готовым preview.

  • Hero title
  • CTA buttons
  • Image/background fields

Как использовать

  1. 1

    Откройте hero preset с готовыми полями для первого экрана.

  2. 2

    Переименуйте заголовок, описание, CTA и изображение под ваш шаблон.

  3. 3

    Экспортируйте WP-шаблон+CSS или PHP-регистрацию для темы.

acf_add_local_field_group([
  'key' => 'group_acf_hero_section',
  'title' => 'ACF Hero Section',
  'fields' => [
    // fields from generator
  ],
]);

Вопросы по ACF Hero Section

Для чего нужен этот генератор?

Нужно управлять hero-блоком из админки WordPress без правки шаблона.

Что получится на выходе?

Поля для первого экрана: заголовок, описание, CTA и изображение hero.

Можно ли доработать поля после генерации?

Да, генератор дает стартовую структуру. Поля можно переименовать, удалить, вложить или дополнить перед экспортом.
