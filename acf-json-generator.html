Как использовать

  1. 1

    Откройте генератор с готовой группой Content Fields и активной вкладкой JSON.

  2. 2

    Переименуйте группу и поля под свой тип записи, шаблон или страницу.

  3. 3

    Скачайте или скопируйте JSON и перенесите его в ACF JSON workflow проекта.

{
  "key": "group_content_fields",
  "title": "Content Fields",
  "fields": [
    {
      "key": "field_hero_title",
      "label": "Hero title",
      "type": "text"
    }
  ]
}

Вопросы по ACF JSON генератор

Для чего нужен этот генератор?

Нужен переносимый JSON для ACF без ручного редактирования.

Что получится на выходе?

Content Fields preset, активная JSON-вкладка, стабильные field keys и проверяемая структура полей.

Можно ли доработать поля после генерации?

Да, генератор дает стартовую структуру. Поля можно переименовать, удалить, вложить или дополнить перед экспортом.
