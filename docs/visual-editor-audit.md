# Visual Editor Audit

Короткий рабочий аудит визуального редактора ACF-генератора.

## Что было слабым

- Размер и толщина шрифта были доступны не везде: часть элементов имела контролы, но CSS превью продолжал использовать жесткие значения.
- fallback visual editor и production renderer расходились по возможностям: один показывал базовые карточки, другой давал более полный production preview.
- У заголовков, подписей, значений, кнопок и FAQ не хватало единых типографических настроек: font-size, font-weight, line-height, letter-spacing.
- Из-за этого пользователь мог менять часть внешнего вида, но не видел ожидаемого результата в live preview или HTML/CSS export.

## Что закрыто

- В fallback visual editor добавлены настройки размера, толщины, интерлиньяжа и трекинга для ключевых элементов.
- В production renderer расширены панели для описания, кнопки и FAQ-ответа.
- Preview CSS теперь применяет те же свойства, которые видны в контролах редактора.
- Добавлен smoke-тест `tests/acf-visual-editor-smoke.mjs`, который фиксирует наличие контролов и применение типографики в CSS.

## Дальше

Следующий продуктовый слой для визуального конструктора: inline-редактирование текстов прямо в iframe, управление отступами отдельных полей и сохранение пресетов оформления.
