Предустановка под запрос «acf field group generator» открывает группу Content Fields с текстовыми, медиа и повторяемыми полями.

  • Group title, key и location rules

Открыть preset
Что входит

Field group с понятными ключами и правилами вывода

Страница ведет в базовый preset группы: group key, field names, location rules и стартовые поля уже собраны для PHP или JSON export.

  • Group title, key и location rules
  • Text, image и repeater fields
  • PHP и JSON export tabs

Как использовать

  1. 1

    Откройте генератор с готовой группой Content Fields.

  2. 2

    Переименуйте группу и поля под свой тип записи, шаблон или страницу.

  3. 3

    Выберите PHP или JSON вкладку и перенесите структуру в проект WordPress.

acf_add_local_field_group([
  'key' => 'group_acf_field_group_generator',
  'title' => 'ACF Field Group генератор',
  'fields' => [
    // fields from generator
  ],
]);

Вопросы по ACF Field Group генератор

Для чего нужен этот генератор?

Нужно быстро собрать field group и не ошибиться в ключах и правилах показа.

Что получится на выходе?

Content Fields preset с group key, field names, location rules, базовыми полями и PHP/JSON export.

Можно ли доработать поля после генерации?

Да, генератор дает стартовую структуру. Поля можно переименовать, удалить, вложить или дополнить перед экспортом.
