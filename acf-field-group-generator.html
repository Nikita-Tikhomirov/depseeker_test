Что входит

Страница закрывает конкретный низкочастотный запрос и ведет в генератор

Пользователь получает понятный ответ, видит состав полей и сразу переходит в инструмент с подходящей предустановкой. Это снижает трение между SEO-трафиком и первым действием.

  • PHP и JSON export

Как использовать

  1. 1

    Откройте генератор с готовой предустановкой под этот тип ACF-структуры.

  2. 2

    Переименуйте группу и поля под свой тип записи, шаблон или страницу.

  3. 3

    Скопируйте PHP или JSON и перенесите в проект WordPress.

acf_add_local_field_group([
  'key' => 'group_acf_field_group_generator',
  'title' => 'ACF Field Group генератор',
  'fields' => [
    // fields from generator
  ],
]);
